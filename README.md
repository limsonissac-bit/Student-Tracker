# Student-Tracker

#  Student Mastery & Analytics Tracker

A real-time academic dashboard built to visualize subject mastery and track study progress. This tool connects a modern frontend UI with a Supabase backend to provide persistent data tracking for students.

##  Live Demo
https://limsonissac-bit.github.io/Student-Tracker/

##  Key Features
- **Dynamic Mastery Visualization:** Interactive SVG progress rings and bar charts that update in real-time as topics are completed.
- **Supabase Integration:** Real-time data fetching and updates for topic status and exam dates.
- **Smart Analytics:** Automatically calculates "Mastery Levels" (e.g., Excellent, Good, Bad) based on completion ratios.
- **Subject Filtering:** High-level overview of progress across multiple subjects (Maths, Physics, etc.).
- **Mobile Responsive:** Modern "Dark Mode" UI designed for both desktop and mobile productivity.

##  Technical Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
- **Backend-as-a-Service:** Supabase (PostgreSQL)
- **Deployment:** GitHub Pages
